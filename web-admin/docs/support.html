FAQ

Comment enregistrer un vol ?

Appuyez sur le bouton "+" sur l'écran principal, puis "Démarrer un vol". L'application enregistrera automatiquement votre trajet GPS.

Comment synchroniser avec Apple Watch ?

Assurez-vous que les deux appareils sont connectés au même compte iCloud. La synchronisation se fait automatiquement.

Comment exporter mes données ?

Allez dans Réglages > Export des données. Vous pouvez exporter en format GPX ou CSV.

L'application fonctionne-t-elle hors ligne ?

Oui, l'enregistrement GPS fonctionne sans connexion internet. Les cartes peuvent nécessiter une connexion pour le téléchargement initial.
