# App Store Screenshots

## Required Sizes

### iPhone
- **6.9" (iPhone 16 Pro Max)**: 1320 x 2868 px
- **6.7" (iPhone 15 Pro Max)**: 1290 x 2796 px
- **6.5" (iPhone 11 Pro Max)**: 1284 x 2778 px
- **5.5" (iPhone 8 Plus)**: 1242 x 2208 px

### Apple Watch
- **Ultra**: 410 x 502 px
- **Series 10 (46mm)**: 396 x 484 px
- **Series 10 (42mm)**: 352 x 430 px

## File Naming Convention

Screenshots should be named with a number prefix for ordering:
- `01_home.png`
- `02_flight_recording.png`
- `03_statistics.png`
- `04_wings.png`
- `05_watch.png`

## How to Take Screenshots

### From Simulator
1. Run app in simulator: `Cmd + R` in Xcode
2. Navigate to the screen you want
3. Take screenshot: `Cmd + S`
4. Screenshots saved to Desktop

### From Real Device
1. Press Side Button + Volume Up simultaneously
2. Find in Photos app
3. AirDrop to Mac

## Recommended Screenshots

1. **Home/Dashboard** - Main flight list
