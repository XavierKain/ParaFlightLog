Politique de confidentialité

Dernière mise à jour : 28 décembre 2025

Introduction

ParaFlightLog ("nous", "notre", "l'application") respecte votre vie privée. Cette politique explique comment nous collectons, utilisons et protégeons vos données personnelles.

Données collectées

L'application collecte les données suivantes, stockées uniquement sur votre appareil :

  • Données de localisation GPS : pour enregistrer vos trajets de vol
  • Données de vol : durée, altitude, vitesse, distance parcourue
  • Informations sur vos équipements : voiles, sellettes (saisies manuellement)
  • Photos : si vous choisissez d'ajouter des photos à vos voiles ou vols

Stockage des données

Toutes vos données sont stockées localement sur votre appareil et synchronisées via iCloud si vous l'avez activé. Nous n'avons pas accès à vos données personnelles.

La synchronisation Apple Watch utilise uniquement la communication locale entre vos appareils.

Partage des données

Nous ne partageons, ne vendons ni ne transmettons vos données personnelles à des tiers.

Si vous choisissez d'exporter vos données (fichiers GPX, CSV), vous êtes responsable de leur utilisation ultérieure.

Services tiers

L'application peut utiliser :

  • Apple Maps / MapKit : pour afficher les cartes (soumis à la politique de confidentialité d'Apple)
  • iCloud : pour la synchronisation de vos données entre appareils (optionnel)

Vos droits

Vous pouvez à tout moment :

  • Accéder à toutes vos données depuis l'application
  • Exporter vos données
  • Supprimer vos données en désinstallant l'application
  • Désactiver la localisation dans les réglages iOS

Sécurité

Vos données sont protégées par les mécanismes de sécurité d'iOS et d'iCloud, incluant le chiffrement des données.

Modifications

Nous pouvons mettre à jour cette politique de confidentialité. Les modifications seront publiées sur cette page avec une nouvelle date de mise à jour.
